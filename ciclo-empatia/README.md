# Ciclo Empatia (Lunae)

Aplicativo mobile (React Native) e API (FastAPI) para compartilhamento de ciclos de empatia entre casais.

**Link do repositório:** https://github.com/Vortexyweb/ciclo_empatia

## Descrição
Plataforma que permite casais registrarem momentos de empatia, trocarem mensagens de apoio e visualizarem gráficos de evolução emocional ao longo do tempo.

## Tecnologias
- Frontend: React Native, Expo
- Backend: FastAPI, PostgreSQL
- Autenticação: JWT
- Deploy: Docker, Docker‑Compose

## Como rodar localmente
1. Clone o repositório
2. Configure o banco de dados PostgreSQL
3. Rode `docker-compose up --build`
4. Acesse o app pelo Expo ou o backend em `http://localhost:8000`
